# Домашнее задание №1  
## CLI-инструмент для получения информации о репозитории GitHub

# Описание
Консольная программа на Go для получения информации о репозиториях GitHub через официальный API.

# Запуск

Соберите программу и запустите с URL репозитория
```bash
./github-info https://github.com/username/reponame
```

# Вывод
Программа возвращает информацию: имя репозитория, описание, количество звёзд, количество форков, дата создания
